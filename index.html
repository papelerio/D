¿Qué más se puede hacer con JSON?

Los JSON pueden almacenar mucha más información que simples tiempos. Aquí te muestro todo el potencial que tienen para crear experiencias de juego ricas y personalizables.

Múltiples Niveles de Dificultad

Define fácil, medio y difícil con diferentes velocidades y patrones.

Tipos de Notas y Patrones

Notas simples, sostenidas, deslizantes y más para crear patrones complejos.

Personalización Visual

Cambia skins, fondos y efectos visuales para cada canción o nivel.

Sistema de Puntuación Avanzado

Define diferentes valores para cada tipo de nota y combo.

Eventos Especiales

Cambios de velocidad, efectos visuales y sonoros en momentos específicos.

Metadatos de Canciones

BPM, offset, información del artista y más para una sincronización perfecta.

Estructura JSON Avanzada para Juegos Rítmicos

  • metadata: Información de la canción (BPM, offset, título, artista)
  • difficultyLevels: Múltiples niveles de dificultad
  • notes: Array de notas con tipo, tiempo, duración y posición
  • events: Eventos especiales (cambios de velocidad, efectos)
  • scoring: Sistema de puntuación personalizado
  • visuals: Personalización de apariencia

Múltiples Niveles de Dificultad

Con JSON puedes definir diferentes niveles de dificultad para adaptarte a todo tipo de jugadores.

{ "song": "Canción Ejemplo", "artist": "Artista Desconocido", "bpm": 120, "offset": 50, "difficultyLevels": { "easy": { "speed": 1.0, "noteDensity": 0.7, "maxSimultaneousNotes": 1, "accuracyThreshold": 150 }, "medium": { "speed": 1.2, "noteDensity": 0.85, "maxSimultaneousNotes": 2, "accuracyThreshold": 100 }, "hard": { "speed": 1.5, "noteDensity": 1.0, "maxSimultaneousNotes": 3, "accuracyThreshold": 75 }, "expert": { "speed": 1.8, "noteDensity": 1.2, "maxSimultaneousNotes": 4, "accuracyThreshold": 50 } } }

Tipos de Notas y Patrones Complejos

JSON permite definir diferentes tipos de notas para crear patrones complejos y variados.

{ "notes": [ { "type": "tap", "time": 1000, "position": 0, "score": 100 }, { "type": "hold", "time": 2500, "duration": 800, "position": 2, "score": 150 }, { "type": "slide", "time": 4000, "positions": [1, 2, 3], "duration": 1000, "score": 200 }, { "type": "simultaneous", "time": 6000, "notes": [ {"position": 0, "score": 100}, {"position": 3, "score": 100} ], "score": 250 } ] }

Personalización Completa

JSON permite personalizar todos los aspectos visuales y de juego para una experiencia única.

{ "visuals": { "background": { "image": "background.jpg", "color": "#1a2a6c", "animation": "parallax" }, "notes": { "tap": { "color": "#2196F3", "texture": "circle.png", "size": 30 }, "hold": { "color": "#4CAF50", "texture": "rectangle.png", "size": [30, 100] } }, "effects": { "perfect": {"color": "#FFEB3B", "particles": 10}, "good": {"color": "#4CAF50", "particles": 5}, "miss": {"color": "#F44336", "particles": 3} } }, "scoring": { "perfect": 100, "good": 50, "bad": 20, "miss": 0, "comboMultiplier": 0.1, "maxComboBonus": 500 }, "audio": { "tapSound": "tap.wav", "holdSound": "hold.wav", "backgroundMusic": "song.mp3", "volume": 0.8 } }
